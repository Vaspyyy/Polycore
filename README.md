**Polycore** is an experimental native geometric arena shooter about movement, builds, evolution, and controlled chaos.

The project is currently in early private development. The goal is to create a polished singleplayer-first arena game with fast combat, persistent progression, smart AI opponents, and deep tank evolution.

## Planned Features

* Native desktop performance
* Smooth arena combat
* Progression-based unlocks
* Expanded evolution paths
* AI opponents with distinct behavior styles
* Replayable singleplayer modes
* Future modding and custom content support

## Status

Polycore is currently **closed source** while the core systems are being designed, rewritten, broken, rebuilt, and probably over-scoped.

Public builds, technical notes, and source access may happen later once the project is stable enough to show without causing psychic damage.

## Tech

Built with Rust, Bevy, and WGPU.
